Data Types in Java
1. Primitive Data Type
	a. byte 	= Size = 1 byte(8  bit) = Default value = 0 = Range = -128 to 127
	b. short	= Size = 2 byte(16 bit) = Default value = 0 = Range = -32,786 to 32,767
	c. int		= Size = 4 byte(32 bit) = Default value = 0 = Range = -2,147,483,648 to 2,147,483,647
	d. long 	= Size = 8 byte(64 bit) = Default value = 0L 
	e. float  	= Size = 4 byte(32 bit) = Default value = 0.0f
	f. double	= Size = 8 byte(64 bit) = Default value = 0.0d 
	g. char		= Size = 2 byte(16 bit) = Default value = blank space
	h. boolean 	= Size = JVM dependent  = Default value = false
2. Non-Primitive, Reference Type, Object Type
	a. Class
	b. Interface
	c. Array
	e. Collections 
	f. String
	g. Wrapper classes

Create New Project
Employee Salary Processing System

Create a Java program that calculates final salary
Inputs
	1. EmployeeType(F = Full Time, P = Part Time)
	2. Experience
	3. Base Salary
Rules
	1. Full Time
		a. Experience > 10 then bonus 20%
		b. Experience > 5 then bonus 10%
		c. Else then bonus 5%
	2. Part Time
		a. Experience > 5 then bonus 8%
		b. Else then bonus 5%

Additional conditions
	Tax
		a. Salary > 1,00,000 then tax will be 10%
		b. Salary > 50,000 then tax will be 5%
		c. Else tax will be 2%

Note : Use Switch Case and Nested If
